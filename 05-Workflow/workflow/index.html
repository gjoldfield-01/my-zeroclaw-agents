    Workflow — AI Thought Leadership Companion

    Turning ideas into social posts.

    Overview

    Field Value
    Name AI Thought Leadership Companion
    Trigger Message via Telegram (voice, text, image, or link)
    Input Unstructured thoughts and ideas
    Output LinkedIn-ready post + draft variations + hashtags

    Step-by-Step Flow

    [DO] Step 1 — Capture & Process Input

    • Transcribe voice notes into text
    • Clean and structure raw input
    • Identify key elements: topic, tone, core idea

    [DO] Step 2 — Extract Insight & Intent

    • Determine underlying theme (leadership, AI, creativity, etc.)
    • Identify target audience (leaders, innovators, creatives)
    • Define core message and angle

    [DECIDE] Step 3 — Validate Direction with User

    Present extracted insight and ask for confirmation or adjustment.

    "This looks like a leadership reflection on X — proceed?"

    [DO] Step 4 — Generate Multiple Post Drafts

    Create 2–3 variations:

    • Reflective — human, thoughtful
    • Professional — clear, leadership-focused
    • Punchy — short, high-engagement

    [DECIDE] Step 5 — Select Draft or Request Edits

    • User chooses version or requests refinement
    • Option to switch tone: "more poetic" / "more direct"

    [DO] Step 6 — Optimise Selected Post

    • Improve hook and opening line
    • Structure for readability (short paragraphs, flow)
    • Add hashtags and optional call-to-action
    • Ensure tone matches user voice (intelligent, human, credible)

    [DECIDE] Step 7 — Final Approval

    • User selects preferred draft or requests refinement
    • Decide platform: LinkedIn (MVP) — Instagram under consideration

    [DO] Step 8 — Store or Prepare Output

    • Save draft to queue or return ready-to-post version
    • Option to schedule or mark as "post now"

    Output Specification

    Output Detail
    Final post LinkedIn-ready
    Draft variations 2–3 options
    Hashtag set 3–5 relevant tags

    Out of Scope

    • Fully autonomous posting without human approval
    • Long-form blog writing or articles
    • Multi-platform formatting (e.g. Instagram vs LinkedIn)

    Design Principles

    Principle Meaning
    Human-in-the-loop Every post approved before publishing
    Insight-first Thinking partnership, not content generation
    Modular Designed to evolve into multi-agent systems
    Low-friction Capture ideas instantly, in any format
