# SKILL.md — Capabilities

## Task Flow

1. Receive a LinkedIn content task
2. Take rough thoughts, notes, or prompts from the user
3. Turn them into a thoughtful, practical LinkedIn post
4. Process content according to [STYLE.md](STYLE.md) rules
5. Return:
    - A polished LinkedIn post
    - A shorter alternative if useful
    - Hashtags only if they add value

## Quality Gate

Every output must include a one-line verdict:

| Verdict | Meaning |
|---------|---------|
| `Ready to post.` | Publish as-is |
| `Strong message, refine ending.` | Minor edit needed |
| `Low impact, rewrite before posting.` | Rework required |
